# Book Recommendation Website

## Описание проекта
Это веб-сайт с описанием книг и системой рекомендаций.  
Пользователи могут просматривать книги, читать описание и получать рекомендации.

## Технологии
В проекте использованы следующие технологии:

- React
- Express.js
- PostgreSQL
- Node.js
- Vite

## Возможности сайта
- Просмотр списка книг
- Просмотр информации о книге
- Система рекомендаций
- Работа с базой данных

## Установка проекта

1. Склонировать репозиторий

git clone https://github.com/username/project-name.git

2. Перейти в папку проекта

cd project-name

3. Установить зависимости

npm install

4. Запустить сервер

npm run dev

## База данных

В проекте используется PostgreSQL.  
База содержит:

- 25 пользователей
- более 200 книг
